Fire Classes

Types of Fires

Class A: Combustible materials (cloth, paper, wood, etc.)

Class B: Flammable and combustible liquids (gasoline, alcohol, oil-based paints, etc.)

Class C: Electrical

Class D: Combustible metals (magnesium, titanium, etc.)

Class K: Oils and fats


Note: fire-extinguishers are marked with what class fire they can be used on - ONLY USED FIRE-EXTINGUISHERS WITH THE CORRESPONDING FIRE (some extinguishers can put out more than one class; check to see what classes they can put out).
